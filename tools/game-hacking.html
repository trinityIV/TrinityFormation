Outils Essentiels

IDA (Interactive DisAssembler)

Le désassembleur et débogueur le plus puissant pour l'analyse de jeux.

Versions :

  • IDA Free 8.3
    • Support limité aux architectures x86
    • Pas de décompilateur
    • Télécharger IDA Free
  • IDA Pro
    • Support multi-architectures
    • Décompilateur Hex-Rays
    • Debugging avancé
    • Acheter IDA Pro

Scripts IDA Python Utiles :

# Recherche de fonctions virtuelles import idaapi import idautils def find_vtables(): for seg in idautils.Segments(): if ".rdata" in idc.get_segm_name(seg): for ea in idautils.Functions(seg, idc.get_segm_end(seg)): if idaapi.get_func(ea).flags & FUNC_VIRTUAL: print(f"Virtual function at {hex(ea)}") # Analyse des chaînes def find_game_strings(): strings = idautils.Strings() for s in strings: if "health" in str(s) or "ammo" in str(s): print(f"Found game string at {hex(s.ea)}: {str(s)}")

Cheat Engine

Outil puissant pour la modification de mémoire en temps réel.

Fonctionnalités principales :

  • Scanner de mémoire
  • Débogueur
  • Injecteur de DLL
  • Table d'adresses
  • Télécharger Cheat Engine

Script CE exemple :

[ENABLE] alloc(newmem,1024) label(returnhere) label(originalcode) label(exit) newmem: push eax mov eax,[health_addr] cmp eax,100 jge exit mov eax,100 mov [health_addr],eax pop eax originalcode: mov eax,[ebp+08] push ebx exit: jmp returnhere "game.exe"+123456: jmp newmem nop returnhere:

Fonctionnalités :

  • Scripting en Python
