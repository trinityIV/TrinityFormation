CVE-2025-1001
Description Détaillée

Buffer overflow vulnerability in GDK Pixbuf image loading allows remote attackers to execute arbitrary code via a specially crafted image file.

Impact Technique
  • Exécution de code arbitraire
  • Crash de l'application
  • Corruption mémoire
Versions Affectées
  • GDK < 3.24.38
  • GTK < 4.10.5
Références
  • Rapport d'analyse
  • Commit correctif
  • PoC (privé)

Code Vulnérable
static void
gdk_pixbuf_load_image(GdkPixbuf *pixbuf, const guchar *buffer, gsize size)
{
    // Code vulnérable - pas de vérification de taille
    memcpy(pixbuf->pixels, buffer, size);
}
Patch Proposé
static gboolean
gdk_pixbuf_load_image(GdkPixbuf *pixbuf, const guchar *buffer, gsize size)
{
    if (size > pixbuf->allocated_size)
        return FALSE;
    
    memcpy(pixbuf->pixels, buffer, size);
    return TRUE;
}
