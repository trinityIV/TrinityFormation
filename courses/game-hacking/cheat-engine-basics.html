1. Introduction

Qu'est-ce que Cheat Engine ?

Cheat Engine est un outil d'analyse de mémoire open source qui permet d'examiner et de modifier les données en mémoire des applications en cours d'exécution. Bien que son nom suggère une utilisation pour la triche dans les jeux, c'est avant tout un outil éducatif puissant pour comprendre :

  • La gestion de la mémoire
  • Les structures de données
  • Le débogage en temps réel
  • L'analyse de code

2. Installation

Téléchargement et Configuration

  1. Visitez le site officiel : www.cheatengine.org
  2. Téléchargez la dernière version stable
  3. Exécutez l'installateur en mode administrateur
  4. Suivez les étapes d'installation en évitant les logiciels tiers proposés
Attention : Certains antivirus peuvent détecter Cheat Engine comme une menace. C'est un faux positif dû à la nature de l'outil.

4. Types de Scan

Méthodes de Scan

Recherche une valeur exacte en mémoire. Utilisé quand vous connaissez la valeur précise à trouver.

Exemple : Recherche de la valeur 100 (points de vie)

Trouve les valeurs qui ont changé depuis le dernier scan.

5. Exercices Pratiques

Premier Scan

Suivez ces étapes pour votre premier scan de mémoire :

  1. Lancez le tutoriel intégré de Cheat Engine
  2. Connectez-vous au processus du tutoriel
  3. Effectuez un scan de type "Exact Value"
  4. Modifiez la valeur trouvée
Conseil : Commencez toujours par le tutoriel intégré pour vous familiariser avec l'interface.

6. Fonctionnalités Avancées

Pointeurs et Structures

Les pointeurs sont essentiels pour suivre des valeurs dont l'adresse change :

  • Recherche de pointeurs avec le Pointer Scanner
  • Création de structures pour organiser les données
  • Utilisation des offsets pour naviguer dans la mémoire

Assembleur et Débogueur

Cheat Engine intègre un débogueur puissant :

  • Visualisation du code assembleur
  • Définition de points d'arrêt (breakpoints)
  • Injection de code
  • Analyse des appels de fonction

Scripts Lua

Automatisez vos analyses avec des scripts Lua :

-- Exemple de script Lua simple
function scanForHealth()
  local value = 100
  local scanType = 4 -- Exact Value
  local valueType = 4 -- 4-byte value
  
  createMemScan()
  memScan.firstScan(scanType, valueType, 1, value, nil, 0, 0xFFFFFFFF, "", 0, 0)
  memScan.waitTillDone()
  
  local foundList = createFoundList(memScan)
  foundList.initialize()
  
  return foundList.Count
end
